- download node
    - https://nodejs.org/en/
    - reboot after install
    - node --version

- npm init (with authorized shell)
    ```
    npm init
    ```
    it will make package.json


- hello.js
    Execute it in cmd bash(not powershell)
    ```
    node hello.js
    ```

- expressjs
    Execute it in the project folder.
    ```
    npm install express
    ```

- server.js
    ```
    node server.js
    ```

    http://localhost:4000/hello
